## Что было выполнено

## Запуск

1. Устанавливаем зависимости из [requirements.txt](requirements.txt).
2. Выполняем миграции и заполняем базу данных:
    ```bash
     python manage.py migrate
     python manage.py loaddata ..\db.json
    ```
